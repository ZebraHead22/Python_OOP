### Задача 1. Система мониторинга производства с автогенерацией отчётов

**Темы:**  
- Классы и объекты, `__init__`, атрибуты класса/экземпляра  
- Наследование, полиморфизм, `super()`  
- Магические методы: `__str__`, `__repr__`, `__eq__`, `__len__`  
- Менеджеры контекста (`__enter__/__exit__`)  
- Обработка исключений (пользовательские исключения)  
- Работа с файлами (CSV, JSON) и построение графиков (`matplotlib`)

**Описание:**  
Разработать систему для мониторинга работы производственных станков. Каждый станок имеет тип, серийный номер, время работы, количество произведённых деталей и журнал событий (поломки, обслуживание).  
- Реализовать иерархию классов: `Equipment` (базовый), `CNCMachine`, `Conveyor`, `RobotArm`.  
- Добавить методы: `run(hours)` – симулирует работу с вероятностью поломки; `service()` – обслуживание; `get_efficiency()` – расчёт эффективности.  
- Использовать `__enter__/__exit__` для автоматической записи лога работы в файл при входе/выходе из контекста.  
- Реализовать класс `ProductionLog`, который агрегирует данные за смену и сохраняет отчёт в JSON, а также строит график эффективности по часам (используя `matplotlib`).  
- Создать пользовательское исключение `MachineOverheatError`, которое выбрасывается при превышении температуры (имитировать случайно).  
- Написать демонстрационный скрипт: создать несколько станков, запустить их в разных контекстах, сгенерировать отчёт.

### Задача 2. Библиотека для работы с временными рядами с поддержкой кэширования и дескрипторов

**Темы:**  
- `@property`, `@classmethod`, `@staticmethod`  
- Магические методы: `__getitem__`, `__setitem__`, `__iter__`, `__next__`  
- Дескрипторы (протокол `__get__/__set__`)  
- Абстрактные классы (`abc.ABC`, `@abstractmethod`)  
- Работа с файлами (чтение CSV, парсинг дат)  
- Построение графиков (`matplotlib`)

**Описание:**  
Создать класс `TimeSeries`, который представляет временной ряд (дата, значение).  
- Использовать дескриптор `PositiveFloat` для проверки, что значения неотрицательны.  
- Реализовать `__getitem__` и `__setitem__` для доступа по дате или индексу.  
- Сделать класс итерируемым (`__iter__`, `__next__`) – при итерации возвращаются пары (дата, значение).  
- Добавить `@property` для получения статистики (среднее, медиана, максимум).  
- Создать абстрактный класс `Transformer` с методом `transform(series)`, от него унаследовать конкретные преобразования: `NormalizeTransformer`, `LagTransformer`, `DiffTransformer`.  
- Реализовать класс `CachedTimeSeries`, который наследуется от `TimeSeries` и переопределяет методы доступа с использованием `__slots__` и кэширования результатов (например, кэш вычислений статистики).  
- Предоставить методы для загрузки из CSV и сохранения в CSV.  
- Построить график исходного и преобразованного рядов (все преобразования применить цепочкой).
